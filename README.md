# Stack

- Frontend: React/Vite
- Backend: Node.js, Express, Socket.io
- Database: PostgreSQL

# How to Run

while in the casino_app directory:

`npm install` to install dependencies

`npm run dev` to start client server and DB server

`npm run db:stop` to stop db when you are done

That's it!
